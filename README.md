### 卷积层1：
- 输入尺寸为16×15。
- 卷积核尺寸为3×3×1×10（3×3空间卷积核，10个卷积核）。
- 输出尺寸为14×13×10×1，空间尺寸变为14×13，通道数为10。
- 激活函数：ReLU。
### 卷积层2：
- 卷积核尺寸为3×3×10×1（3×3×10空间卷积核，1个卷积核）。
- 输出尺寸为12×11×1x1。
- 激活函数：ReLU。
### 全连接层1：
- 输人尺寸为132×1（来自卷积层的输出打平）。
- 权重大小为132×10（全连接）。
- 输出尺寸为10×1，每个节点输出10个值。
- 激活函数：ReLU。
### 全连接层2：
- 输人尺寸为10×1。
- 权重大小为10×1。
- 输出尺寸为1，表示最终的分类输出（目标或非目标）。
